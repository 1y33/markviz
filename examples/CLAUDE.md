# Notes directory

When generating markdown in this directory, follow the markviz authoring skill at `../.claude/skills/markviz/SKILL.md`.

Key conventions:
- `[[wikilinks]]` between related notes
- `flashcards` block at the end of conceptual notes
- KaTeX `$...$` and `$$...$$` for math
- ` ```mermaid ` blocks for diagrams
- Single `# H1` title per note
- `#topic` tags

For research notes, flashcards, or paper digests, the skill has dedicated templates — apply them.
